About

A free web-based crm program created by Peter Petropoulos. It can be yours right now, simply download it on the github page. The program is open source, and you can use it for any purpose, personal or commercial.

You can try the program features at the preview page. Connect using admin user and admin password.

It includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization.
